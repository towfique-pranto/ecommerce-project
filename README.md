# E-Commerce React Project

### [Live Demo](https://ecommerce-project-react-1qs0.onrender.com)

This repository contains my personal implementation of the projects from the [React Course](https://github.com/towfique-pranto/react-course) by [SuperSimpleDev](https://github.com/SuperSimpleDev). It serves as a record of my learning journey and progress through the curriculum.

A modern, responsive e-commerce application built with **React** and **Vite**. This project features a full shopping flow, from product browsing to order tracking.

---

## Features

### Home Page

- **Dynamic Product Grid:** Browse a wide range of products with real-time price formatting.
- **Interactive Cart:** Add products to your cart with instant feedback.
- **Search & Filter:** Quickly find products using the search functionality.

### Checkout

- **Order Summary:** View a detailed breakdown of your cart items.
- **Delivery Options:** Choose between different shipping methods with dynamic price updates.
- **Real-time Validation:** Ensuring all order details are correct before placement.

### Orders & Tracking

- **Order History:** View your past purchases with detailed summaries.
- **Live Tracking:** Monitor the status of your orders with a visual progress bar.
- **Buy Again:** Easily re-order items from your history.

---

## Tech Stack

### Frontend

- **React 19:** Utilizing the latest features for optimal performance and developer experience.
- **Vite:** Extremely fast build tool and development server.
- **React Router 7:** Handling complex navigation and routing with ease.
- **Axios:** Managing API requests to the backend.
- **Dayjs:** Simple and lightweight library for date formatting.

### Backend (AI-Generated)

The backend was entirely generated by AI to provide a working server and database for the application.

- **Node.js & Express:** Providing a robust API layer for data persistence and business logic.
- **Sequelize ORM:** Handling database interactions and schemas.
- **SQLite:** Reliable local database for storing product and order information.

## Installation & Setup

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/towfique-pranto/ecommerce-project.git
    ```

2.  **Install dependencies:**

    ```bash
    # Navigate to the project root
    cd ecommerce-project

    # Install client dependencies
    cd client
    npm install

    # Install server dependencies
    cd ../server
    npm install
    ```

3.  **Run the application:**

    ```bash
    # Start the backend server (from /server)
    npm run dev

    # Start the frontend dev server (from /client)
    npm run dev
    ```

## Deployment

This project is configured for **Render**.
The build process automatically compiles the React frontend, moves the production `dist` folder into the server directory, and serves the entire application from a single Node.js instance.
